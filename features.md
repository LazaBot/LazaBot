# LazaBot Features

LazaBot offers a powerful set of features designed to help Discord servers stay safe, organized, and engaging.  
Each feature is built with performance, reliability, and ease of use in mind.

### AutoMod
Automatically detects and handles spam, raids, and abusive behavior using configurable rules to keep servers protected 24/7.

### ActionTag
Every moderation action performed by LazaBot is assigned a unique **ActionTag (Case ID)**, making moderation transparent and easy to track.

### AccessNode
An advanced permission handling system that validates user permissions, bot permissions, and role hierarchy before executing moderation actions, preventing errors and crashes.

### SmartReplies
An intelligent auto-response system that detects trigger phrases or common questions and instantly replies with pre-configured answers, no human intervention needed.

### Event-Based AutoRoles
Automatically assign roles to members based on server events such as joining, verification completion, or other configured triggers.

### Reaction Roles
Allow members to assign roles to themselves by reacting to messages, reducing manual role management for moderators.
